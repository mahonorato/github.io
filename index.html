Linguagem R

R é um ambiente computacional e uma linguagem de programação que vem progressivamente se especializando em manipulação, análise e visualização gráfica de dados.

Linguagem SAS

O software SAS foi, inicialmente, desenvolvido na Universidade Estadual da Carolina do Norte e, atualmente, é uma das plataformas proprietárias mais utilizadas para análise de dados.

Este site é um repositório de scripts, descrições de comandos e documentações de apoio utilizadas diariamente em minhas atividades de engenharia de dados.

Leia o Manual é uma base de conhecimento mantida por Marcos Honorato (marcos.honorato@jhionline.net)
