# Dados Estruturados

SAS trabalha com dados estruturados e não estruturados. Como fonte de dados estruturados podem ser utilizadas planilhas, arquivos separados por vírgula, tabelas de bancos de dados como Oracle, MS Acces, Terada, Hadoop éntre outros, além do formato proprietário da SAS, as tabelas sas7bdat.

As tabelas do SAS possuem as seguintes características:

  • Nomes:
    • São limitados a 32 caracteres;
    • Devem começar por "_" ou uma letra;
    • Podem ser definidos em maiusculo, minúsculo ou combinação de ambos.

Para evitar eventuais problemas quando tralhando com diferentes ambientes, preferivelmente, devem ser utilizadas somente letras, números ou "_" após o primeiro caracter. ☺

  • Tipos:

    • Numéricos
      • digitos 0-9;
      • sinal de menos (-);
      • ponto decimal; e
      • Notação científica (E).
    • Caracteres
      • Letras;
      • Números;
      • Caracteres especiais; e
      • "Vazio".
    • Datas
      • As datas são representadas numericamente em relação a data 01/jan/1960;
      • As datas anteriores são armazendas como números negativos; e
      • as posteriores como positivos;

  • Comprimento:

    • Numéricos: 8 bytes
    • Caracteres: 1 - 32.767 bytes
