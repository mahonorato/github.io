# Explorando e Validando Dados

Em estatística, a análise exploratória de dados (AED) é uma abordagem à análise de conjuntos de dados de modo a resumir suas características principais, frequentemente com métodos visuais. Um modelo estatístico pode ou não ser usado, mas primariamente a AED tem como objetivo observar o que os dados podem nos dizer além da modelagem formal ou do processo de teste de hipóteses.Wikipedia

O SAS fornece uma série de "procedures" que facilitam o trabalho de análise exploratória dos dados, como CONTENTS, PRINT, MEANS, UNIVARIATE e FREQ.

(Os exemplos criados com a tabela SASHELP.AIR)

CONTENTS

Exibe um resumo sobre o arquivo ou tabela carregada.

Mais informações sobre CONTENTS na documentação oficial

data tbl_air;
	set sashelp.air;
run;

proc contents data=tbl_air;
run;

PRINT

Cria uma lista com todas as colunas e linhas de dados:

Mais informações sobre PRINT na documentação oficial

data tbl_air;
	set sashelp.air;
run;

proc print data=tbl_air;
run;

MEANS

Cria umum sumário de estatísticas para as colunas com dados numéricos. A procedure MEANS apresenta a frequencia de ocorrência, a média, o desvio padrão e os valores máximos e mínimos.

Com este sumário é possível determinar se existem valores nulos numa determinada coluna e a existência de valores inválidos. Mais informações sobre MEANS na documentação oficial

data tbl_air;
	set sashelp.air;
run;

proc means data=tbl_air;
run;

UNIVARIATE

Também cria um sumário com estatísticas, desta vez mais relacionadas à distribuição do dados.

Com este sumário é possível observar os cinco menores e maiores valores. Mais informações sobre UNIVARIATE na documentação oficial

data tbl_air;
	set sashelp.air;
run;

proc univariate data=tbl_air;
run;

FREQ

Cria uma tabela de frequencia para cada coluna da tabela.

Na tabela de frequencias é possível identificar problemas como valores inconsistentes. Mais informações sobre FREQ na documentação oficial

data tbl_air;
	set sashelp.air;
run;

proc freq data=tbl_air;
run;
